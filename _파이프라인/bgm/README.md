# BGM 폴더

이 폴더는 비어있어야 합니다 (.gitignore로 mp3/wav 제외).

## BGM 설치 방법

옵션 1 — 심볼릭 링크 (권장):
```bash
ln -s ~/Downloads/bgm $(dirname "$0")
```

옵션 2 — 직접 복사:
```bash
cp ~/Downloads/bgm/*.mp3 .
```
